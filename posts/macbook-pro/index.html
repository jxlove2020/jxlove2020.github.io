맥북프로 초기화

맥북프로 초기화 하기 ( Option + command + R )

macOS 버전에 따라서 설치가 안되거나 실행이 안되는 프로그램이 있을 수 있으니
가능한 최신 버전을 설치하는게 좋은 것 같아요.
