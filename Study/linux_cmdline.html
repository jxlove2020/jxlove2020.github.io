리눅스 커맨드라인 명령어

  • ls - 디렉토리의 파일리스트를 출력
  • cat - 파일의 내용을 출력
  • head - 앞에 10줄만 출력
  • tail - 뒤에 10줄만 출력
  • wc - 워드 갯수 출력 ( 옵션 -ㅣ 라인수 출력 )
  • nl - 줄 번호 출력
  • sort - 정렬
  • uniq - 중복 제거 sort 와 함께 사용 (-d 중복된 내용만 출력, -u 중복되지 않은 내용만 출력, -i 대소문자 무시)
  • cut - 텍스트 자르기
  • tr - 텍스트 찾아 바꾸기
  • sed - 줄단위 작업
  • awk 강력한 스크립팅
