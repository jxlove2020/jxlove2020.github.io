# 마크다운(MarkDown)

### 제목
```
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
```

*이텔릭*

### 인용문
> 인용문
>> 중첩 인용문

<br>

### 수평선
```
3개 이상 입력하면 수평선 이 나옵니다.

--- 하이픈
*** 애스터리크
___ 언더바
```

# Visual Studio

### Ctrl + 함수 클릭 : 함수로 이동
### Ctrl + '-' , Ctrl + Shift + '-' : 앞 뒤로 탐색
### Alt + 키보드 방향 (위, 아래) : 위 아래로 이동
### Alt 키 + 블럭 지정 후 글자 수정 : 여러 줄 의 글자 수정 가능
### Ctrl + Shift + F : 문자열 모조리 찾기
### Ctrl + R + R : 함수명 변경
### Ctrl + K + F : 자동정렬
### Tab 키 활용 : 코드 자동완성 
### Ctrl + K + C : 주석
### Ctrl + K + U : 주석해제
### 모든 참조 찾기 ( 환경 - 키보드 - 문자열을 포함하는 명평 표시 - 모든참조찾기 검색 후 - 바로가기 키 누르기 - 키보드 눌러 설정 (Ctrl + K + R or Ctrl + Shift + G) - 할당 )
